## The Password Illusion

Most people believe they are using strong passwords.

Capital letter? ✔
Number? ✔
Symbol? ✔

Secure? Not necessarily.

Patterns like:

-   Name@123
-   India@2024
-   Welcome@1
-   ChildName2015

are among the most predictable password formats in the world.

Hackers don't guess randomly.
They test patterns and humans are predictable.

------------------------------------------------------------------------

## A Real Case: Domino's India Data Breach (2021)

In 2021, Domino's India suffered a massive data breach.
Reports indicated that data of over 180 million users was exposed,
including names, phone numbers, email addresses, and order details.

Now imagine this scenario:

A user used the same password for:

-   Domino's account
-   Gmail
-   Social media
-   Online banking

If that password was exposed, attackers wouldn't need to hack anything.
They would simply try the same email-password combination elsewhere.

This method is called **credential stuffing**.

One leak.
Multiple compromised accounts.

# Password Security: Do's & Don'ts

## ✅ DO

-   Use 12--16+ character passwords
-   Use unique passwords for every account
-   Enable two-factor authentication (2FA)
-   Use a trusted password manager
-   Prioritize email security

## ❌ DON'T

-   Reuse passwords
-   Use names or birthdays
-   Add "@123" and assume it's secure
-   Share OTPs
-   Click urgent-looking suspicious links

------------------------------------------------------------------------

## What Actually Makes a Strong Password?

A strong password:

✔ Is long (12--16+ characters)
✔ Is random
✔ Is unique for each account
✔ Avoids personal information

Example:

Instead of:
Harveen123

Use something like:
Blue\$River!84Sun

Even better, let a password manager generate it.
